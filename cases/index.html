Data source for COVID-19 Cases

1. Data Organization

  • This data is organized as follows:

    • County level data located in us-counties.csv

    • State level data located in us-states.csv

    • National level data located in us.csv

2. Preliminary Look at the Data

library(tidyverse)

counties <- read_csv("nyt/us-counties.csv")

states <- read_csv("nyt/us-states.csv")

us <- read_csv("nyt/us.csv")
